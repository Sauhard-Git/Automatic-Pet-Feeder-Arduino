# 📖 Overview

The **Automatic Pet Feeder** is a DIY automation project designed to dispense pet food automatically in a controlled manner.

Built using an **Arduino Uno**, **Servo Motor**, and an **IR Sensor**, the system detects a pet's presence and releases a measured quantity of food into the bowl.

This project demonstrates the practical application of **Embedded Systems**, **Automation**, and **Electronics** to solve a real-world problem.

# 🔌 Circuit Connections

## 🦾 Servo Motor

| Servo Pin | Arduino Pin |
|-----------|-------------|
| Signal | D9 |
| VCC | 5V |
| GND | GND |

### 📡 IR Sensor

| IR Pin | Arduino Pin |
|---------|-------------|
| OUT | D2 |
| VCC | 5V |
| GND | GND |

# ⚙️ Working Principle

```text
🐾 Pet Approaches
        │
        ▼
📡 IR Sensor Detects Motion
        │
        ▼
🧠 Arduino Processes Signal
        │
        ▼
⚙️ Servo Opens Food Gate
        │
        ▼
🥣 Food Dispensed into Bowl
        │
        ▼
🔒 Servo Closes the Gate
```
